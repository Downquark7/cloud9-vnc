VNC
===

Running X11 in a Cloud9 workspace.

![Screen Shot](screenshot.png)

Installation
------------

    cd /home/ubuntu/workspace/
    git clone https://github.com/Downquark7/cloud9-vnc.git
    cd /home/ubuntu/workspace/cloud9-vnc/
    sudo apt-get update
    ./install.sh

Running
-------

    ./run.sh
    
or use the `X11` runner.
